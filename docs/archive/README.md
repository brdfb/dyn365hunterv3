# Archived Documentation

This folder contains archived documentation from completed phases.

## Archive Format

Files should be archived with date prefix:
- `YYYY-MM-DD-filename.md`

## Archive Rules

1. **When to Archive**: When a phase is complete and no longer actively referenced
2. **What to Archive**: 
   - Completed TODO lists
   - Completed phase documentation
   - Old prompts (if not actively referenced)
3. **Keep Active**: Only current phase documentation in `active/`
