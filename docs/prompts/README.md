# Important Prompts

This folder contains important prompts and conversation snippets that should be preserved for reference.

## Format

Each prompt file should follow this format:

```markdown
# Prompt Title

**Date**: YYYY-MM-DD
**Context**: Brief description
**Status**: Active/Archived

## Prompt

[The actual prompt]

## Response Summary

[Key points from the response]

## Actions Taken

- [ ] Action 1
- [x] Action 2 (completed)
```

## Archive Policy

- Move to `archive/` when no longer actively referenced
- Keep only current/active prompts in root
- Use date prefix for archived prompts: `YYYY-MM-DD-prompt-name.md`
