# Important Prompts & Decisions

This folder contains important prompts, conversation snippets, and **architectural decisions** that should be preserved for reference.

## Format

Each prompt file should follow this format:

```markdown
# Prompt Title

**Date**: YYYY-MM-DD
**Context**: Brief description
**Status**: Active/Archived

## Prompt

[The actual prompt]

## Response Summary

[Key points from the response]

## Actions Taken

- [ ] Action 1
- [x] Action 2 (completed)
```

## File Types

### Prompts
- Important conversation snippets
- Setup instructions
- Key discussions

### Decisions
- Architectural decisions (marked with `[DECISION]` in title)
- Technology choices
- Design decisions
- Format: `YYYY-MM-DD-decision-name.md`

## Archive Policy

- Move to `archive/` when no longer actively referenced
- Keep only current/active prompts in root
- Use date prefix for all files: `YYYY-MM-DD-prompt-name.md` or `YYYY-MM-DD-decision-name.md`
